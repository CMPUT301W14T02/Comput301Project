Help: Browsing a Specific Comment and its Replies

Sorting

To sort the list of comments select the "Sort by:" drop-down menu. Here is a list of sorting functions to choose from and an explaination for what each of them does:

Default
The default sorting method for the comments. It displays the newest comments that are close to your location at the top of the list, while older comments that are farther away are displayed near the bottom.
Date
Displays the comments in the order that they were created, with newest comments at the top of the list and oldest at the bottom.
Picture
Displays comments that have pictures attached at the top of the list and comments that don't have pictures attached at the bottom.
My Location
Displays comments that are close to your current location determined by GPS at the top of the list and comments posted farthest from you at the bottom of the list. If you do not have GPS turned on it will be sorted by a default location of (0,0) or your last known location.
Other Location
Displays comments that are close to a custom location of your choosing at the top of the list and comments posted farthest from the custom location at the bottom of the list.
Faves
Displays comments with the most favorites at the top of the list and comments with the least favorites at the bottom.

Viewing Selected Comment Details

At the top of the screen the selected comment is displayed, along with an attached photo if it has one and the author's username.

Browsing Replies

Replies are comments that are responses to a different comment. You can see the list of all the replies to the selected comment on the app from this screen. Selecting a reply allows you to view that specific comment and its replies.

Editing the Selected Comment

If the selected comment was created by a user with the username you are currently logged in as, you are able to edit that comment from this page. To edit the comment select "Edit Comment" from the action bar menu.

Replying to the Selected Comment

You can create a reply to the selected comment by pressing the "Reply" button.

Adding the Selected Comment to Your Favorites

To add the comment to your favorites press the "Add to Favorites" button. A message will be displayed on the screen if the request was successful. Adding the comment to your favorites allows you to view it from the "Favorites" option of the main menu. Adding the comment to your favorites also increases the number of Faves that comment has, effecting its position when sorting by Faves.

Following the Author of the Selected Comment

To follow the user that created the selected comment press the "Follow" button. A message will be displayed on the screen if the request was successful. Following the author allows you to view all of their comments from the "Followed User's Comments" option of the main menu.
